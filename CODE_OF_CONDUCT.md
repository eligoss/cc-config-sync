# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Please read the full text at the link above. In summary, we are committed to providing a welcoming and inclusive experience for everyone.

## Reporting

If you experience or witness unacceptable behavior, please open an issue or contact the maintainer directly.
